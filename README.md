# BEGINNER-ML-CHATBOT
 This is a beginner-friendly Python chatbot that recommends skincare products based on skin type, skin tone, and skin concerns.
A. Features
1.Rule-based logic
2.Beginner friendly
3.No machine learning required
4. Easy to extend

B. How It Works
1. User enters skin type
2. User enters skin tone
3. User selects skin concerns
4. Chatbot recommends skincare routine and products

C. How to Run
python chatbot.py
